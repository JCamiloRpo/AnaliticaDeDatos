
Para trabajo de modelamiento matematico se desarrollaron dos soluciones:

1)Realizar el recomendador por los metodos vistos en clase de modelamiento matematico, haciendo uso del derivadas
parciales, funcion de error regresion logistica y por el gradiente de esta misma función.
(Archivo "Recomendador_Modelamiento.py que utiliza los .csv)

2)Realizar el recomendaor por medio de metodos de inteligencia artificial para predecir variables futuras (en este
caso saber si a una persona le gusta o no una pelicula), usando modelos como arboles de decisiones, KNN (vecinos 
cercanos) y redes neuronales (BackPropagation y DeepLearning). (Archivo "Recomendador_IA.py" que utiliza los .xlsx)
Para que este archivo funcione satisfatoriamente debe tener instalado las siguientes librerias
	- tensorflow
	- keras
	- pydotplus

para instalarlas debe abrir "Anaconda Prompt" y debe ingresar por medio del comando "cd" a la ruta donde se encuentra
la carpeta "Anaconda3". Una vez ubicado allí debe dijitar los siguientes comandos
	- python -m pip install tensorflow
	- python -m pip install keras
	- python -m pip install pydotplus
